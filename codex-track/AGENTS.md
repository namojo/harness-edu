# AGENTS.md — 설비 모니터링 대시보드 프로젝트 규칙

## 불변 규칙 (모든 에이전트 공통)
- 대시보드 코드는 `contracts/`에 정의된 API 계약만 바라본다. 시뮬레이터 내부 직접 접근 금지.
- 단위는 표준 스키마(SET v1)에서 통일한다: 시간=초, 수율=%, 수량=장. 단위 변환은 어댑터(정규화 계층)의 책임이다.
- 수치를 임의로 지어내지 않는다. 계약·사전에 없는 상태값이나 필드는 [미확인]으로 격리 보고한다.
- 정적 배포(GitHub Pages) 가능해야 하므로 빌드 도구 없는 순수 HTML/CSS/JS로 작성한다.
- 코드 수정 후에는 브라우저 콘솔 에러 0건을 확인한다.

## 산출물 위치
- 계약 문서: `contracts/` (SET v1 스키마, E10 상태 사전, API 명세)
- 구현: `index.html`, `css/`, `js/`
- QA 리포트: `qa/`
