분임조마다 용어, 은어, 문체, 표기 형식이 제각각인 보고서를

   매핑으로 표준 항목(계획수량, 생산수량, 양품, 불량, 가동시간,

   - 비고·특이사항의 은어(예: 빵꾸, 데나오시, 콤프, L/S)는 표준용어로 치환

     별도 보고
4. 문체 통일: 비고와 특이사항은 개조식 평서체("~함", "~발생")로

   확인하고, 불일치 시 원본 값을 유지한 채 [데이터 이상] 플래그를 다세요.
