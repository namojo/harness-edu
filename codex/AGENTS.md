# AGENTS.md — 전략마케팅팀 트렌드 보고서 작성 원칙

## 불변 규칙 (모든 에이전트 공통)
- 모든 수치·주장에는 출처(매체/기관명)와 날짜를 붙인다. 출처 없는 수치는 쓰지 않는다.
- 확인되지 않은 내용은 추측으로 메우지 말고 "[확인 필요]"로 표시한다.
- 조사 노트는 research/, 최종 보고서는 report/ 에 저장한다.
- 보고서 형식·문체·출처 표기는 trend-report 스킬의 템플릿을 따른다.
- 경쟁사 관련 서술은 공개 정보만 사용하고, 단정 대신 근거와 함께 서술한다.
- 웹 검색 시 1년 이상 지난 자료는 '과거 맥락'으로만 인용하고 최신 동향과 구분한다.

## 산출물 위치
- 분야별 조사 노트: research/{분야}.md
- 최종 보고서: report/{YYYY-MM}.md 또는 report/trend-report.md
- 검증 결과: report/fact-check.md
